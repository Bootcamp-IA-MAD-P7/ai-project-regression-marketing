Resultados:

Conclusión:

El dataset presenta una calidad estructural muy elevada.

#### Identificación preliminar de variables

Se clasificaron las variables según su naturaleza:

- Temporales.
- Binarias.
- Geográficas.
- Operativas.
- Publicitarias.
- Económicas.

Se identificaron variables con alta cardinalidad:

- ad_id
- ad_group_id
- campaign_id

Estas variables requerirán evaluación posterior para determinar si aportan valor predictivo o generan ruido.

#### Evaluación preliminar del target

Variable candidata seleccionada:

```text
revenue
```

Resultados principales:

- Media: 475,71
- Mediana: 0,00
- Máximo: 104.733,41
- Distribución altamente asimétrica.

Se identificó una fuerte concentración de campañas sin ingresos y una cola larga de valores elevados.

#### Análisis preliminar de Data Leakage

Se calcularon correlaciones con la variable objetivo.

Variables más relacionadas con Revenue:

| Variable | Correlación |
|-----------|------------:|
| conversions | 0.76 |
| clicks | 0.48 |
| spend | 0.19 |

Hallazgos adicionales:

- month ↔ week → 0.97
- impressions ↔ reach → 0.97

No se detectó leakage directo, aunque varias variables representan información generada durante la ejecución de las campañas y requerirán una definición formal de uso antes del modelado.

#### Conclusiones del Dataset Assessment

Se concluyó que:

- El dataset es adecuado para un proyecto de regresión.
- La calidad general es elevada.
- Revenue es un candidato sólido a variable objetivo.
- Será necesario definir formalmente la política de selección de variables y gestión del posible data leakage antes de comenzar el modelado.
