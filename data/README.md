# Data

This folder is intended to store the dataset used in this project.

## Dataset Source

Digital Marketing Performance Dataset  
Source: Kaggle

## Instructions

The dataset is not included in this repository.

To reproduce the project:

1. Download the dataset from Kaggle.
2. Place the CSV file inside this folder.
3. Keep the original filename or update the notebook path accordingly.

## Git Policy

Raw data files should not be committed to the repository unless explicitly required.

Only documentation files such as this README should be tracked by Git.
